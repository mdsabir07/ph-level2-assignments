Vercel
=========
install Vercel globally = npm i -g vercel
login = vercel login
create a josn file on main root directory 'vercel.json' and add the json code like below:
{
    "version": 2,
    "builds": [
        {
            "src": "dist/server.js",
            "use": "@vercel/node"
        }
    ],
    "routes": [
        {
            "src": "/(.*)",
            "dest": "dist/server.js"
        }
    ]
}

now sending to production = vercel --prod
here follow the instruction:
set up and deploy... = yes/y enter
which scope ... your project = y / enter
Link to existing project = n (no if want to new project)
What's your project name = express_server (example and small with small letter)
In which directory.... code located = ./ enter
Do you want to change ... settings = n enter
Detected a repository. connect.... project = n enter
